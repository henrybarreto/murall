# Murall

Murall is simple API for save a get messages.

> Golang training project

## Technologies

- Go
- Gorilla
- MongoDB

## Paths

`localhost:8080/save`
> Save a message in the database

`localhost:8080/`
> Get the last message from the database

## Commands

### Run the API

```bash
go run internal/main
```
